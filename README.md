# vue3-numberlist

Easy Vue3 app to start learning this JS framework! :)

![Image text](https://github.com/lucas-jb/vue3-numberlist/blob/main/.img/image.PNG)


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
